# Meteo App
A little meteo app built in Javascript using DarkSky API

## Getting Started

### Run the App
To run the app, open the index.html with your browser. It's better to open it with a local server.


## Built With

- HTML
- Javascript
- CSS

## Usefull

- [DarkSky API](https://darksky.net)
